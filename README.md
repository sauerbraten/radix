    <article class="markdown-body entry-content" itemprop="mainContentOfPage"><h1>
<a name="radixtrie" class="anchor" href="#radixtrie"><span class="mini-icon mini-icon-link"></span></a>radixtrie</h1>

<p>An implementation of the <a href="http://en.wikipedia.org/wiki/Radix_tree">radix trie data structure</a> in Go</p>

<h2>
<a name="usage" class="anchor" href="#usage"><span class="mini-icon mini-icon-link"></span></a>Usage</h2>

<p>Get the package:</p>

<pre><code>$ go get github.com/sauerbraten/radixtrie
</code></pre>

<p>Import the package:</p>

<pre><code>import (
    "github.com/sauerbraten/radixtrie"
)
</code></pre>

<p>You can use the radixtrie as a key-value structure, where every node's can have its own value (as shown in the exmaple below), or you can of course just use it to look up strings, like so:</p>

<pre><code>trie := radixtrie.New()
trie.Insert("foo", true)
fmt.Printf("foo is contained: %v\n", trie.Find("foo"))
</code></pre>

<h3>
<a name="example" class="anchor" href="#example"><span class="mini-icon mini-icon-link"></span></a>Example</h3>

<p>This example code is taken from the radixtrie_test.go file</p>

<pre><code>package main

import (
    "github.com/sauerbraten/radixtrie"
    "fmt"
)

func main() {
    // create new trie
    trie := radixtrie.New()

    // insert some strings
    trie.Insert("abc", "value 1")
    trie.Insert("a", "value 2")
    trie.Insert("abd", []byte("value 3"))
    trie.Insert("b", 4)

    // print trie structure, the parameter sets the initial level of indentation
    trie.Print(0)

    // delete some strings, even strings not contained
    trie.Delete("c")
    trie.Delete("b")
    trie.Delete("ab")

    // print again, notice the changes:
    // 'b' is gone, 'ab' is no longer an end note, means it is no longer contained as a string
    trie.Print(0)

    // use Find() to check if a string is contained in the trie
    fmt.Printf("'a' holds: %v\n", trie.Find("a"))
    fmt.Printf("'c' holds: %v\n", trie.Find("c"))
    fmt.Printf("'abd' holds: %v\n", trie.Find("abd"))
}
</code></pre>

<p>This example should print the following:</p>

<pre><code>''  end: &lt;nil&gt;
'a'  end: value 2
    'b'  end: &lt;nil&gt;
        'c'  end: value 1
        'd'  end: [118 97 108 117 101 32 51]
'b'  end: 4
''  end: &lt;nil&gt;
    'a'  end: value 2
        'b'  end: &lt;nil&gt;
            'c'  end: value 1
            'd'  end: [118 97 108 117 101 32 51]
'a' holds: value 2
'c' holds: &lt;nil&gt;
'abd' holds: [118 97 108 117 101 32 51]
</code></pre>

  <h2>Markdown Cheat Sheet</h2>

  <div class="cheatsheet-content">

  <div class="mod">
    <div class="col">
      <h3>Format Text</h3>
      <p>Headers</p>
      <pre>
# This is an &lt;h1&gt; tag
## This is an &lt;h2&gt; tag
###### This is an &lt;h6&gt; tag</pre>
     <p>Text styles</p>
     <pre>
*This text will be italic*
_This will also be italic_
**This text will be bold**
__This will also be bold__

*You **can** combine them*
</pre>
    </div>
    <div class="col">
      <h3>Lists</h3>
      <p>Unordered</p>
      <pre>
* Item 1
* Item 2
  * Item 2a
  * Item 2b</pre>
     <p>Ordered</p>
     <pre>
1. Item 1
2. Item 2
3. Item 3
   * Item 3a
   * Item 3b</pre>
    </div>
    <div class="col">
      <h3>Miscellaneous</h3>
      <p>Images</p>
      <pre>
![GitHub Logo](/images/logo.png)
Format: ![Alt Text](url)
</pre>
     <p>Links</p>
     <pre>
http://github.com - automatic!
[GitHub](http://github.com)</pre>
<p>Blockquotes</p>
     <pre>
As Kanye West said:

> We're living the future so
> the present is our past.
</pre>
    </div>
  </div>
  <div class="rule"></div>

function fancyAlert(arg) {
